http와 https의 차이

```md
http는 우선 하이퍼텍스트 문서들을 네트워킹 할 때 사용하는 프로토콜,

```

cors

```md
크로스 오리진 리소스 쉐어링
동일한 리소스에서 서로 요청을 보낼 수 있는 기본적으로 정해져야 하는 규칙으로 알고 있음.
프로토콜, 포트 번호, 도메인, 이렇게 세가지 모두 같아야 cors정책에 맞다고 알고 있음.
```

  url과 같은 형식으로 서버로 요청을 보내는데 서버가 이를 보고 비교해야 판단이 가능한데

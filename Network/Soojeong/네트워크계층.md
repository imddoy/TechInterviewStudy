### 1. 네트워크 계층은 OSI 모델의 3계층

OSI 7계층 중 **3계층이 네트워크 계층(Network Layer)**이에요.

이 계층의 목적은 단 하나예요.

> 데이터를 출발지에서 목적지까지 어떻게 보낼지 결정하는 계층

단순히 `보낸다`가 아니라

- 어디로 보내야 하는지
- 어떤 길로 가야 하는지
- 다음 장비가 무엇인지

를 결정해요.

### 2. 네트워크에서 실제로 어떤 일이 일어날까?

네트워크 계층에서 벌어지는 일은 눈에 안 보이지만 인터넷 통신의 핵심이에요.

**실제로 네트워크 계층에서 하는 일**

1. 목적지 IP 주소를 확인한다.
2. 라우팅 테이블을 참고한다.
3. 최적의 경로를 선택한다.
4. 패킷을 다음 라우터로 보낸다.
5. 목적지 네트워크에 도착할 때까지 반복한다.

즉, 컴퓨터는 “무작정 보내지 않아요.”

항상 경로 계산 후 전송해요.

### 3. 네트워크 계층의 기능

**1) 논리 주소 지정 (IP 주소 부여)**

네트워크 계층에서는 **IP 주소**를 사용해 장비를 구분해요.

MAC 주소는 물리 주소, IP 주소는 위치 개념의 논리 주소예요.

- MAC 주소: 기기 자체
- IP 주소: 기기가 현재 연결된 위치

그래서 IP는 바뀔 수 있어요.

**2) 라우팅 (Routing)**

라우팅은 목적지까지 갈 **길을 결정**하는 작업이에요.

라우터는 항상 다음 질문을 해요.

> "라우팅 테이블을 보니까…
>
> 이 IP는 어디로 보내는 게 제일 빠르지?"

그래서 라우터에는 항상 **라우팅 테이블**이 존재해요.

**3) 패킷 전달 (Forwarding)**

라우팅은 “길을 정하는 과정”이고 포워딩은 “실제로 보내는 과정”이에요.

라우터는

- 패킷 받음
- 주소 확인
- 경로 결정
- 다음 장비로 전송

이 일을 초당 수만 번 반복해요.

**4) 단편화(Fragmentation)**

네트워크마다 한 번에 전송할 수 있는 최대 크기가 달라요.

이걸 MTU(Maximum Transmission Unit)라고 하는데, 만약 패킷이 너무 크면

→ 쪼개서 보내요 `Fragmentation`

### 4. 네트워크 계층 장비: 라우터

네트워크 계층의 대표 장비는 라우터예요.

라우터 하는 일:

- 네트워크를 연결한다.
- 경로를 계산한다.
- 패킷을 전달한다.

스위치는 2계층, 라우터는 3계층

### 5. TCP/IP에는 네트워크 계층이 없다?

없다 ! TCP/IP 모델에는 "네트워크 계층"이라는 이름이 없어요.

대신 다른 이름이 존재해요.

### 6. TCP/IP의 인터넷 계층(Internet Layer)

OSI의 네트워크 계층 역할은 TCP/IP에서는 인터넷 계층이 해요.

즉,

> 이름만 다르고, 역할은 거의 같아요.

**OSI vs TCP/IP 계층 비교**

OSI 모델

- 물리 계층
- 데이터 링크 계층
- **네트워크 계층**
- 전송 계층
- 세션
- 표현
- 응용

TCP/IP 모델

- 네트워크 접근 계층
- **인터넷 계층**
- 전송 계층
- 응용 계층

### 7. 인터넷 계층의 주요 프로토콜

- IP
  논리 주소 설정 + 패킷 전달 담당
- ICMP
  에러 발생 시 알림용
  - 목적지 없음
  - TTL 초과
  - 네트워크 장애
    (Ping도 ICMP 사용)
- ARP
  IP 주소를 이용해 MAC 주소를 알아내는 프로토콜
  > "이 IP 누구?"
- IGMP
  멀티캐스트 그룹 관리

### 8. 네트워크 계층은 신뢰성을 보장하지 않아요

네트워크 계층은

- 전달은 해주지만
- 도착 보장은 안 한다
- 오류 복구도 안 한다

"잃어버렸으면?" → 전송 계층(TCP)이 처리

> 책임은 → 전송 계층
>
> 이동만 → 네트워크 계층

### 9. IPv4 vs IPv6 간단 비교

IPv4

- 32비트
- 주소 부족

IPv6

- 128비트
- 주소 무한급
- IPSec 기본 지원

### 10. 한 줄 요약

네트워크 계층은

> 주소를 보고 길을 찾고 패킷을 다음으로 넘기는 계층이에요.

그리고

> TCP/IP에서는 네트워크 계층이란 이름 대신 **인터넷 계층**이 같은 일을 해요.

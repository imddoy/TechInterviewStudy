# 메모리 모의면접 ( 한수정  )

Q : 논리주소와 물리주소의 차이점, 왜 분리하는지

A : 논리주소와 물리주소를 나눈다는 글을 봤는 것 같은데 이유를 모르겠다

---

Q : 고정분할방식과 가변분할방식의 차이점

A : 고정분할방식은 메모리 크기를 고정된 방식으로 분할을 하는데 해당 공간이 고정되어있다보니 프로세스와 크기가 맞아야하고 들어가야한다고 해도 남는 공간이 없어야함

가변분할방식은 그 크기를 변경할 수 있도록해서 더 좋은 효율을 냄

---

Q : 쓰레싱이 발생을 하면 안좋은데 이를 완화할 수 있는 방법?

A : 쓰레싱은 필요한 메모리에 맞는 프로세스를 계속 가져오고 내려보내는 스와핑이 발생했을때 발생하는 문제점. 방법은 모르겠음

---

Q : 페이지와 프레임의 차이?

A : 고정된 단위로 나누는 것이 페이지이고 프로세스를 나누는것이 프레임이다

---

Q : 페이지 테이블의 역할? 각 엔트리에 어떤 정보?

A : 페이지 테이블에 대한 정리를 안함

---

Q : 자바스크립트에서 메모리 누수가 발생하는 경우 세가지?

A : 모르겠습니다

→ 자바스크립트에서 변수를 선언해놓고 사용하지 않았을 때 그 변수가 쌓이면 메모리 누수가 생길 수 있음

---

Q : TLB 가 뭔지, 페이지 테이블과의 관계가 무엇인지 모르겠음

A : 모르겠다

---

Q : 페이징과 세그멘테이션에서 각각 어떤 단편화가 발생하는지?

A : 페이징과 세그멘테이션 둘다 고정 크기로 나누지만, 세그멘테이션은 논리적인 단위로 나누는 것에서 차이가 있고, 외부단편화를 해결한다고 읽었던 것 같다
